Create Md Files
------------------
`bin/updatehtdocs.js` renders md files (prev/next links) and a toc (index.md) into htdocs path.

the source md files are in `mds/`.

special files: 
   * `mds/TOC` for section order.
   * `skins/` templates for final mds
